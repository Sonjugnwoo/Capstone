# C++/MFC 기반 실시간 객체 탐지 시스템 (캡스톤 디자인 프로젝트)

YOLOv3 딥러닝 모델을 C++/MFC로 구현한 실시간 객체 탐지 시스템입니다.

캡스톤 디자인 (졸업 프로젝트)  
YOLOv3 80개 클래스 실시간 탐지  
MFC GUI 완성도 높은 UI/UX  
실시간 처리 (25-35fps 목표)  
머신비전 포트폴리오 핵심 프로젝트

## 주요 기능
- 실시간 카메라 영상 처리
- YOLOv3 객체 탐지 (COCO 80 클래스)
- 바운딩 박스 + 클래스 라벨 표시
- FPS 표시 및 성능 모니터링
- 이미지/비디오 파일 입력 지원
- MFC 기반 직관적 UI
- 탐지 결과 로그 저장
- 전동킥보드 인도 감지 + 자동 메일 알림 (16프레임 연속)

## 기술 스택
| 항목         | 버전 / 정보                |
|--------------|----------------------------|
| OS           | Windows 10 / 11           |
| IDE          | Visual Studio 2019        |
| 언어         | C++ (C++11 이상)          |
| 프레임워크   | MFC                       |
| 딥러닝       | YOLOv3 (Darknet 기반)     |
| 컴퓨터비전   | OpenCV 3.2                |
| GPU 가속     | CUDA                      |
